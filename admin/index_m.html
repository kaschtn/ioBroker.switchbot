Help
To get your API credentials:
  1. 1. Open SwitchBot app
  2. 2. Go to Profile > Preferences
  3. 3. Tap App Version 10 times
  4. 4. Tap Developer Options
  5. 5. Tap Get Token

Note: You need SwitchBot app version 6.14 or later to get the secret key.

Supported Devices
  • Physical Devices: Bot, Curtain, Lock, Meter, Plug, Color Bulb, Humidifier, Motion Sensor, Contact Sensor, etc.
  • Infrared Devices: TV, Air Conditioner, Light, Fan, etc.
Configuration Notes
  • Poll Interval: Minimum 10 seconds (10000ms). Lower values may exceed API rate limits.
  • Cloud Service: Must be enabled in the SwitchBot app for BLE devices to work via API.
  • API Limits: 10,000 requests per day per account.
